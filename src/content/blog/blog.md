## Welcome to My Blog! 🚀

Hello and welcome to my very first blog post! This blog was built using **Astro** 
and deployed on **Cloudflare Workers** as part of my web development lab. 
I'm excited to share my journey as I learn to build modern web applications.

## What I'm Learning

In this lab, I am exploring some amazing modern web technologies:

- **Cloudflare Workers** — Serverless code that runs at the edge globally
- **Astro** — A fast, content-focused web framework
- **Vite** — A lightning-fast frontend build tool
- **React** — A popular UI component library
- **Cloudflare D1** — Cloudflare's serverless SQL database

## What is Astro?

Astro is a modern web framework designed for building fast, 
content-rich websites like blogs, portfolios, and documentation sites. 
What makes Astro special is that it sends **zero JavaScript to the browser 
by default**, making your site incredibly fast.
